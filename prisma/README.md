# Prisma siting

to generate Prisma client

```js
    npx prisma init --datasource-provider sqlite
```

to build a tables in database

```js
     npx prisma db push
```

to generate tables

```js
    npx prisma generate
```

prisma studio ++localhost:5555++

```js
    npx prisma studio
```
